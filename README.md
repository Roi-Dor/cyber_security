# Cybersecurity Portfolio

This repository contains practical cybersecurity tools built to demonstrate and learn core concepts relevant to Cloud Security, Containerization, and AI-driven defense.

## Projects

| Project | Tech Stack | Key Concepts |
| :--- | :--- | :--- |
| **[Container Scanner](./Rogue_Container_Scanner)** | Bash, Docker | Least Privilege, Runtime Security, JSON Parsing |
| **[Cloud Toxic Combo Scanner](./Instance_Scanner)** | Python, Boto3 | Cloud Posture Management (CSPM), AWS IAM, Exposure Analysis |
| **[AI Log Analyzer](./Log_Analyzer)** | Python, OpenAI API | GenAI for Security, Log Analysis, Automated Remediation |
